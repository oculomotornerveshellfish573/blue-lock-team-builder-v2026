## 🛠️ How to Use the Builder

Open the application after the installation finishes. The main screen shows a soccer pitch layout. 

### Creating a Squad
Select the "New Squad" button to start. Drag player icons from the sidebar onto the pitch. The application highlights valid positions for each player. When you place a player in a specific role, the system updates the team statistics.

### Using Smart Mode
Enable "Smart Mode" to speed up your process. This feature automatically fills the pitch with players based on their skill ratings. You can lock specific players on the field while the system fills the remaining spots.

### Random Generation
Select the "Randomize Team" button to test new combinations. This tool ignores current selections and builds a fresh squad of eleven players. Use this when you need inspiration for a new lineup.

## 💡 Troubleshooting

If you encounter issues, review these common fixes.

**The application locks up during startup.**
Close the application and start it again. Ensure your computer has the latest Windows updates installed.

**The roster does not save.**
Check your folder permissions. Ensure the application has permission to write files to your documents folder. Occasionally, antivirus software prevents the app from saving. Add the application to your trusted list in your antivirus settings.

**The visuals look blurry.**
Navigate to the "Settings" menu within the application. Select "Graphics" and adjust the scale settings to match your monitor resolution.
